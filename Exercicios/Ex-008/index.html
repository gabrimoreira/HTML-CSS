Prinicipais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito e um em italico usando tag b e i (não semanticas)

Por ser não recomendado utilzar tags não semaânticas, deve-se utilizar as tags STRONG(para negrito) e EM(para itálico)

Podemos criar um texto marcado usando a tag MARK.

Outras formas e semanticas especiais

Texto grande tag BIG e Texto pequeno tag SMALL

Texto deletado tag DEL e Texto sublinhado Tag U(não semântico) Tag INS(Semântica)

Para elevar um numero X2+3 use a tag SUP

Para usar um numero pequeno em baixo H2O use a tag SUB
